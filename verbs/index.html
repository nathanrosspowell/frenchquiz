The Special Verbs

French verbs can be split into a few different groups. These verbs have their own special conjugations, hence, irregular verbs. Learning each one will be a task of memorization!

Lessons

Avoir - To Have

If you wish to have something in French, learn these words.

i       = j'ai        - ( je + ai )
you     = tu as       - ( "two ahh" )
he      = il a
she     = elle a
we      = nous avons
you(p)  = vous avez
they(m) = ils ont
they(f) = elles ont

�tre - To be / I am

This doesn't translate directly because of the different meanings for each of the roles.

i        = je suis
you      = tu es 
he       = il est
she      = elle est
we       = nous sommes 
you(p)   = vous etes
they(m)  = ils sont
they(f)  = elles sont

Aller - To go

How to get around.

i       = je vais
you     = tu vais
he      = il va
she     = elle va
we      = nous allons
you(p)  = vous allez
they(m) = ils vont
they(f) = elles vont
